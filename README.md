ELF Dissection
===============

> by Vito Kurnik

#### Examples of use:

##### Parsing the ELF header

./demo_elf -h demo_elf

##### Parsing the list of sections

./demo_elf -S demo_elf

##### Parsing the symbol table

./demo_elf -s demo_elf

##### Parsing the selected section

./demo_elf -d .text demo_elf
